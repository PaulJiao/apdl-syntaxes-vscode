# APDL README

An extension with rich support for the APDL(ANSYS Parametric Design Languages) language.

## Features

![screenshoot_1](images/screenshoot_1.png)

Support file extensions:
- **.mac**
- **.ans**
- **.ansys.txt**
- **.ans.txt**
- **.ansys**
- **.apdl**
